Windows Password Recovery

Forgot your password to Windows... no problem.
I've included here everything one might need to do it yourself.

By ZeroOneZero

Downloads

  1. Kon-Boot CD

  2. InfraRecorder to burn ISO to CD

  3. Rufus to Burn to USB

Instructions

  1. Download the ISO from above.

  2. Burn the ISO to a blank CD

  3. Turn off your computer, insert disc and power on.

  4. Use the F keys (F2, F9, F10, F11 or F12) while booting to see if you can find the boot menu
    Once you select CD from the menu the program will do the rest.

  5. Once the computer comes back up simply login (with no password). Make sure to change your password and remove the disc.
